# Zenny Space

Zenny Space is a mental health platform built on a Polyglot Microservices architecture.

## Tech Stack
- **Core:** Ruby on Rails (API Mode)
- **AI Agent:** Python (FastAPI + Vertex AI)
- **Database:** PostgreSQL (Cloud SQL) & Firestore
- **Infra:** Google Cloud Run (Serverless)
